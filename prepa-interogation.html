Définition Nucléocapside = Acide nucléique + Capside

2. Multiplication virale

  1. ADSORPTION : protéine virale ↔ récepteur cellulaire spécifique
  2. PÉNÉTRATION (3 modes) :
    • Translocation directe (virus nus)
    • Fusion enveloppe/membrane (virus enveloppés)
    • Endocytose
  3. DÉCAPSIDATION : libération de l'AN viral
  4. RÉPLICATION/TRANSCRIPTION : synthèse des composants viraux
  5. ASSEMBLAGE : auto-assemblage des virions
  6. LIBÉRATION : par lyse (virus nus) ou bourgeonnement (virus enveloppés)

Notion clé PERMISSIVITÉ : capacité d'une cellule à favoriser la multiplication virale (récepteurs + facteurs nécessaires)

Virus de la grippe (Myxovirus)

  • Variation antigénique (surtout type A)
  • Sous-types : 16 HA (H1-H16), 9 NA (N1-N9)
  • Exemples : H3N2 (Hong Kong 1968), H5N1 (aviaire), H1N1 (porc 2009)
  • Transmission : voies respiratoires, mains, objets
  • Prévention : mesures barrières, vaccination complexe

Virus de la rage (Rhabdoviridae)

  • Structure : forme d'obus, ARN sb, enveloppé
  • Transmission : morsure/griffade d'animal infecté
  • Types : citadine (chiens), sylvatique (renards), chiroptères
  • Diagnostic : corps de Negri (nécropsie)
  • Prévention : vaccination animaux + personnes à risque

Poxvirus

  • Variole : éradiquée en 1979 (vaccination)
  • Monkeypox : Afrique Centrale/Ouest, clinique similaire à variole
  • Prévention : virus de la vaccine

Papillomavirus (HPV)

  • Structure : ADN db, icosaédrique, nu
  • Plus de 120 types (30 à tropisme génital)
  • Pathologies :
    • Non oncogènes verrues, condylomes acuminés ("crêtes de coq")
    • Oncogènes types 16, 18, 31, 33 → cancer col utérin (70% cas), cancer anal
  • Facteurs de risque : VIH, partenaires multiples, tabac, non circoncision

🍄 MYCOLOGIE

1. Généralités

  • Mycologie = étude des champignons (mycoses)
  • 70 000 espèces dont ±500 pathogènes pour l'homme
  • Groupes :
    • Filamenteux (moisissures) : hyphes septés ou non
    • Levures : unicellulaires
    • Macromycètes : champignons à chapeau
  • Habitat : sol, air, eau ; saprophytes, commensaux ou pathogènes

2. Caractères généraux

Morphologie

FormeDescriptionExemples

MycéliumEnsemble de filaments

Structure eucaryote

  • Noyau avec membrane
  • Membrane plasmique lipoprotéique
  • Paroi stratifiée (antigènes, résistance)
  • Vacuoles, mitochondries

3. Mycoses superficielles

Dermatophytoses

Agents : Microsporum (17), Trichophyton (22), Epidermophyton floccosum

Sources : anthropophilique, zoophilique, tellurique

Atteintes :

  • Peau : dermatophytie circinée, intertrigos
  • Cuir chevelu : teignes
  • Barbe/moustache : folliculite
  • Ongles : onyxis (bord libre, pieds 80%)

Diagnostic : examen direct (KOH) → filaments + arthrospores ; culture (Sabouraud)

Malasseziose (Pityriasis versicolor)

  • Agent : Malassezia furfur (levure lipophile)
  • Facteurs : chaleur, humidité, peau grasse, grossesse, corticothérapie
  • Lésions : macules hypochromiques squameuses (cou, épaules, thorax)
  • Diagnostic : spores "en grappes" + filaments

Tinea nigra

  • Agent : Hortaea werneckii
  • Distribution : zones tropicales (sol)
  • Lésions : taches brunes/noires (paumes, plantes)
  • Diagnostic : filaments bruns, culture noire

Piedra

4. Mycoses sous-cutanées

Mycétomes

TypeAgentGrainsDistribution
Fongique (eumycétome)Madurella mycetomatisBlancs/noirsZones nord-tropicales
ActinomycosiqueActinomadura, NocardiaFins <1µm avec franges

Clinique : Pied de Madura (tuméfaction → fistules → grains dans le pus)

Diagnostic : examen grains, culture (Sabouraud, Löwenstein)

Chromoblastomycose

  • Agents : champignons noirs dimorphes (Fonsecae, etc.)
  • Contamination : piqûres épines (milieux ruraux)
  • Lésions : papule → nodule → forme verruqueuse (pieds, jambes)
  • Complications : surinfection, éléphantiasis, cancérisation
  • Diagnostic : examen direct, culture, histologie

Mucormycoses tropicales

TypeAgentClinique
BasidiobolomycoseBasidiobolus ranarumNodule sous-cutané → placard dermo-hypodermique
ConidiobolomycoseConidiobolus coronatusGranulome nasal → déformation faciale ("groin de tapir")

Sporotrichose

  • Agent : Sporothrix schenckii (dimorphe)
  • Contamination : cutanée (épines, griffures), inhalation
  • Forme principale : dermato-lymphangitique "en chapelet"
  • Autres formes : pulmonaire, ostéo-articulaire, disséminée
  • Diagnostic : culture, histologie

5. Mycoses profondes

Candidoses

  • Agent : Candida albicans (levure)
  • Facteurs favorisants : ATB, diabète, immunodépression, corticothérapie, macération
  • Formes superficielles : muqueuses (muguet), ongles, peau
  • Formes profondes : septicémies (endogène/exogène)
  • Diagnostic : examen direct (levures bourgeonnantes + pseudo-mycélium), culture

Cryptococcose

  • Agent : Cryptococcus neoformans (levure encapsulée)
  • Facteurs : immunodépression (Sida, cancer)
  • Porte d'entrée : pulmonaire
  • Forme principale : méningo-encéphalite subaiguë
  • Diagnostic : encre de Chine (capsule), culture, sérologie

Pneumocystose humaine

  • Agent : Pneumocystis jirovecii
  • Terrain : immunodéprimés (CD4 <200/mm³)
  • Transmission : inhalation, interhumaine possible
  • Clinique : pneumopathie
  • Diagnostic : recherche dans LBA (coloration spéciale)

Histoplasmose

TypeAgentDistributionClinique
AméricaineH. capsulatumAmériquePulmonaire (aiguë/chronique), disséminée
AfricaineH. duboisiiAfriqueNodules cutanés, abcès osseux

Diagnostic : levures intracellulaires (expectorations, pus) ; culture (dimorphisme) ; sérologie (histoplasmine)

Aspergillose

  • Agent : Aspergillus fumigatus (principal)
  • Habitat : sol, matières organiques, air
  • Formes : aspergillome, allergique, invasive, extrapulmonaire
  • Toxines : A. flavus → aflatoxines (hépatocarcinome)
  • Diagnostic : direct, culture, sérologie, histologie
