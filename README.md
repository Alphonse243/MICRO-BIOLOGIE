# 📚 FICHE DE RÉVISION : MICROBIOLOGIE GÉNÉRALE (L1)

## MODULE I : BACTÉRIOLOGIE GÉNÉRALE

### 1. OBJET DE LA MICROBIOLOGIE
- **Définition** : Étude des micro‑organismes (M‑O) : bactéries, virus, champignons.
- **Microbiologie médicale** : Étude des relations entre M‑O et l’homme (commensalisme, mutualisme, parasitisme).

### 2. MORPHOLOGIE ET STRUCTURE BACTÉRIENNE

#### 2.1. Formes et regroupements
- **Formes** : coques (ronds), bacilles (bâtonnets), vibrions (virgule), spirilles, spirochètes.
- **Regroupements** : diplocoques, streptocoques, staphylocoques (amas), sarcines, « lettres chinoises » (*Corynebacterium*), ramifications (*Actinomyces*).

#### 2.2. Structure de la cellule procaryote (de l’extérieur vers l’intérieur)

| Structure | Caractéristiques | Rôle(s) principal(aux) |
| :--- | :--- | :--- |
| **Capsule** | Polysaccharides, rigide, visible à l’encre de Chine | Adhérence, résistance à la phagocytose (**virulence**) |
| **Couche muqueuse** | Amorphe, déformable | Protection ? |
| **Flagelles** | Longs filaments ondulés (monotriche, lophotriche, péritriche) | **Mobilité**, chimiotactisme |
| **Fimbriae** | Fins, rigides, très nombreux | **Adhésion** aux cellules, formation de **biofilms** |
| **Pili** | Plus longs, moins nombreux (pili sexuels) | **Conjugaison** (transfert d’ADN) |
| **Paroi** | Peptidoglycane (muréine). Distinction fondamentale **Gram + / Gram –** | Protection osmotique, forme, pouvoir pathogène, colorabilité |
| **Membrane plasmique** | Bicouche phospholipidique | Barrière de perméabilité, respiration, synthèse |
| **Cytoplasme** | Contient ribosomes (70S), corps d’inclusion | Synthèse protéique, stockage |
| **Matériel génétique** | **Chromosome** (ADN circulaire unique), **Plasmides** (ADN extrachromosomique) | Information génétique, résistance aux ATB, virulence |

#### 2.3. Détail de la paroi et colorations
- **Bactérie à Gram positif (BGP)** : paroi épaisse, nombreux peptidoglycane, acides teichoïques → violet.
- **Bactérie à Gram négatif (BGN)** : paroi mince, peptidoglycane fin, **membrane externe** avec **lipopolysaccharide (LPS)**.
    - **LPS = endotoxine** : constitué du **lipide A** (toxique), d’un core polysaccharidique et de l’**antigène O** (spécifique).

#### 2.4. Structures spéciales : l’ENDOSPORE
- **Quoi ?** Forme de dormance très résistante (*Bacillus*, *Clostridium*).
- **Structure** : exosporium → tunique sporale → cortex → noyau (protoplaste avec ADN, SASP, DPA‑Ca).
- **Facteurs de résistance** :
    1. **Complexe DPA‑Ca** : déshydrate la spore et stabilise l’ADN.
    2. **Protéines SASP** : protègent l’ADN des UV et de la chaleur.
- **Cycle** : sporulation (formation) → germination (retour à l’état végétatif).

### 3. PHYSIOLOGIE BACTÉRIENNE

#### 3.1. Nutrition et milieux de culture
- **Besoins** : macronutriments (C, H, O, N…), micronutriments (Co, Zn…), facteurs de croissance (vitamines…).
- **Milieux** :
    - **Défini (synthétique)** : composition chimique exacte connue.
    - **Complexe** : à base d’hydrolysats (peptones…).
    - **Sélectif** : inhibe certaines bactéries, favorise d’autres.
    - **Différentiel** : permet de distinguer différents types (ex. coloration des colonies).

#### 3.2. Croissance bactérienne
- **Paramètres** : temps de génération (TG), croissance exponentielle.
- **Courbe de croissance (milieu clos)** : **latence → exponentielle → stationnaire → mort**.
- **Facteurs influençant** :
    - **Température** : psychrophiles, **mésophiles** (pathogènes, 37°C), thermophiles.
    - **pH** : acidophiles, neutrophiles (la plupart), alcalophiles.
    - **Pression osmotique** : **halophiles** (aiment le sel), halotolérants.
    - **Oxygène** : aérobies stricts, anaérobies stricts, anaérobies facultatifs, microaérophiles.

### 4. RELATIONS HÔTE–MICRO‑ORGANISMES ET POUVOIR PATHOGÈNE

#### 4.1. Flore bactérienne normale (FBN)
- **Avantages** : protection (compétition, production de substances toxiques), apports vitaminiques (B, K).
- **Inconvénients** : réservoir de M‑O potentiellement pathogènes (si immunodépression, brèche…).

#### 4.2. Pathogenèse et facteurs de virulence

**A. Facteurs de colonisation**
- **Adhésion** : fimbriae, adhésines, biofilms.
- **Invasion** : invasines (favorisent l’entrée dans les cellules).
- **Compétition** : sidérophores (captent le fer), protéases (clivent les IgA).
- **Mobilité** : flagelles, chimiotactisme.

**B. Facteurs de lésions tissulaires**
- **Exotoxines** (protéines sécrétées) :
    - **Toxines A‑B** : sous‑unité B de liaison, sous‑unité A toxique (ex. choléra, tétanos).
    - **Toxines cytolytiques** : lysent les membranes (ex. hémolysines, leucocidines).
    - **Superantigènes** : hyperstimulation immunitaire massive (ex. toxine du choc toxique).
- **Endotoxine (LPS)** : lipide A, libéré à la lyse des BGN. Effets systémiques : fièvre, inflammation, choc septique, CIVD.
- **Enzymes** : hyaluronidase (dissémination), DNase, etc.
- **Mimétisme moléculaire** : induction de réactions auto‑immunes.

### 5. CONTRÔLE DES MICRO‑ORGANISMES

#### 5.1. Agents physiques
- **Chaleur** : humide (autoclave 121°C – tue les spores), sèche (four 160‑180°C), pasteurisation.
- **Radiations** : UV, gamma.
- **Filtration** : 0,22 µm pour éliminer les bactéries.

#### 5.2. Agents chimiques
- **Stérilisants** : tuent toutes formes de vie (y compris spores). Ex. oxyde d’éthylène (gaz).
- **Désinfectants** : sur surfaces inertes, ne tuent pas forcément les spores.
- **Antiseptiques** : sur tissus vivants.

### 6. ANTIBIOTIQUES ET ANTIBIORÉSISTANCE

#### 6.1. Classification
- **Par cible** : paroi, membrane, synthèse protéique (ribosome 30S/50S), synthèse des acides nucléiques.
- **Par effet** : **bactéricides** (tuent) vs **bactériostatiques** (inhibent la croissance).

#### 6.2. Résistance aux antibiotiques
- **Types** : naturelle / acquise.
- **Mécanismes biochimiques** : inactivation enzymatique, modification de la cible, diminution de perméabilité, efflux actif.
- **Bases génétiques** :
    - **Mutation chromosomique** (sélection).
    - **Transfert de plasmides** (conjugaison) – dissémination rapide.
    - **Transposons** (« gènes sauteurs ») : intégration dans plasmide ou chromosome.

#### 6.3. Évaluation de l’efficacité
- **CMI** (concentration minimale inhibitrice) : empêche la croissance.
- **CMB** (concentration minimale bactéricide) : tue les bactéries.
- **Méthodes** : diffusion (antibiogramme de Kirby‑Bauer) – qualitative ; dilution (milieu liquide ou solide) – quantitative (CMI, CMB).

#### 6.4. Associations d’antibiotiques
- **Pourquoi ?** Synergie, prévention de la résistance (ex. tuberculose), élargissement du spectre, traitement probabiliste.
- **Résultats** : synergie, antagonisme, indifférence.

#### 6.5. Prévention de l’antibiorésistance (3 volets, 10 messages)
1. **Prévenir les infections** : vacciner, transmission croisée, limiter les invasifs.
2. **Mieux utiliser les ATB** : bonnes modalités, savoir dire non, bon traitement initial, savoir arrêter.
3. **Savoir refuser les ATB** : réévaluer à 48 h, traiter l’infection (pas la colonisation), infections bactériennes uniquement.

## MODULE II : VIROLOGIE GÉNÉRALE

### 1. GÉNÉRALITÉS SUR LES VIRUS

#### 1.1. Définition et structure
- **Définition** : agents infectieux **acellulaires**, possédant un seul type d’acide nucléique (ADN ou ARN), parasite intracellulaire obligatoire.
- **Structure (virion)** :
    - **Acide nucléique** : ADN ou ARN, simple brin (sb) ou double brin (db), linéaire/circulaire, segmenté.
        - **ARN positif** : directement traduit en protéines.
        - **ARN négatif** : doit être transcrit en ARN+.
    - **Capside** : coque protéique (protomères). Protège l’AN, antigénique. Symétries : **icosaédrique**, **hélicoïdale**, **complexe**.
    - **Nucléocapside** = AN + capside.
    - **Enveloppe** : bicouche lipidique (provenant de la cellule hôte) + glycoprotéines (gp) virales. Rôle : attachement, fusion, antigénicité. Sensible aux solvants.

#### 1.2. Classification
- Critères : type d’AN, symétrie, enveloppe, taille, hôte, mode de transmission, famille.

#### 1.3. Cycle de multiplication virale
1. **Adsorption** (attachement à un récepteur)
2. **Pénétration** (translocation, fusion, endocytose)
3. **Décapsidation** (libération de l’AN)
4. **Réplication/transcription** (synthèse des composants viraux)
5. **Assemblage** (auto‑assemblage)
6. **Libération** (par lyse ou bourgeonnement)

#### 1.4. Transmission et évolution
- **Modes** : aérienne, vectorielle (arbovirus), féco‑orale, contact direct (sang, sexe, salive), mère‑enfant.
- **Évolution** :
    - **Aiguë** : courte (grippe, Ebola).
    - **Persistante/chronique** : réplication continue (VIH, VHB, VHC).
    - **Latente** : virus présent mais non répliquant, réactivation possible (HSV, VZV).
    - **Lente** : longue incubation puis évolution fatale (prions).

### 2. ÉTUDE DES VIRUS PATHOGÈNES

#### 2.1. Arbovirus (transmis par arthropodes)
- **Exemples** : fièvre jaune (*Flavivirus*), dengue (*Flavivirus*), chikungunya (*Togaviridae*), fièvre de la Vallée du Rift (*Bunyaviridae*).
- **Tableaux** : fièvres hémorragiques, syndromes algo‑fébriles, encéphalites.

#### 2.2. Filovirus (Ebola, Marburg)
- **Caractéristiques** : ARNsb, enveloppé, filamentaire. Réservoir : chauve‑souris.
- **Pathologie** : fièvre hémorragique grave. Transmission interhumaine par liquides biologiques.
- **Souches Ebola** : Zaïre (ZEBOV, +virulente), Soudan, Bundibugyo, Côte d’Ivoire.

#### 2.3. Virus de la grippe (*Orthomyxoviridae*)
- **Caractéristiques** : ARN segmenté, enveloppé. Protéines de surface : **hémagglutinine (HA)** (attachement) et **neuraminidase (NA)** (libération).
- **Variabilité** : **glissement** (mutations ponctuelles, épidémies) et **cassure** (réassortiment génétique, pandémies).
- **Types** : A (pandémies), B, C.

#### 2.4. Virus de la rougeole et des oreillons (*Paramyxoviridae*)
- **Rougeole** : ARN, enveloppé. Transmission aérienne. Immunité durable.
- **Oreillons** : ARN, enveloppé. Transmission aérienne. Complications : orchite, méningite.

#### 2.5. Entérovirus (*Picornaviridae*)
- **Caractéristiques** : ARN, nus. Transmission féco‑orale.
- **Poliovirus** : maladie « iceberg » (peu de paralysies). Vaccin.
- **Coxsackies A/B, Echovirus** : syndromes variés (respiratoire, méningite, mains‑pieds‑bouche…).

#### 2.6. Herpesviridae
- **Caractéristiques** : ADNdb, enveloppé. Propriété clé : **latence**.
- **Alpha‑herpesvirinae** : HSV1 (oral), HSV2 (génital), VZV (varicelle, zona). Latence dans les ganglions nerveux.
- **Beta‑herpesvirinae** : CMV (mononucléose, infections congénitales), HHV6 (roséole). Latence dans monocytes/macrophages.
- **Gamma‑herpesvirinae** : EBV (mononucléose infectieuse, lymphomes), HHV8 (sarcome de Kaposi). Latence dans lymphocytes B.

#### 2.7. Virus de la rage (*Rhabdoviridae*)
- **Caractéristiques** : ARN, enveloppé, forme d’obus. Très fragile.
- **Transmission** : morsure/griffure d’animal infecté (zoonose). Virus neurotrope.
- **Prévention** : vaccination animale et humaine (après exposition).

#### 2.8. Virus des hépatites
| Virus | Famille | Génome | Transmission | Chronicité | Prévention |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **VHA** | Picornavirus | ARN nu | Féco‑orale | Non | Hygiène, vaccin |
| **VHB** | Hepadnavirus | ADN enveloppé | Sang, sexe, mère‑enfant | Oui (cirrhose, CHC) | Vaccin |
| **VHC** | Flavivirus | ARN enveloppé | Sang +++ | Oui (cirrhose, CHC) | Pas de vaccin |
| **VHD** | Virus défectif | ARN enveloppé | Sang, sexe | Oui (si co‑infection VHB) | Vaccin VHB |
| **VHE** | Calicivirus | ARN nu | Féco‑orale | Non | Hygiène |

#### 2.9. Papillomavirus (HPV) (*Papovaviridae*)
- **Caractéristiques** : ADNdb, nu. Tropisme cutanéo‑muqueux.
- **Types** :
    - **Non oncogènes** : verrues, condylomes acuminés (« crêtes de coq »).
    - **Oncogènes** (16, 18, 31, 33) : cancer du col de l’utérus, cancer anal.
- **Transmission** : contact cutané, IST+++.

#### 2.10. Rétrovirus (VIH, HTLV)
- **Caractéristiques** : ARN, enveloppé. Possèdent une **transcriptase inverse** (ARN → ADN).
- **VIH (virus de l’immunodéficience humaine)** :
    - **Structure** : gp120 (attachement au CD4), gp41 (fusion), transcriptase inverse.
    - **Cibles** : lymphocytes T CD4+ via récepteur CD4 et corécepteurs (CCR5, CXCR4).
    - **Cycle** : fixation → fusion → décapsidation → transcription inverse → intégration → réplication → assemblage → bourgeonnement.
    - **Conséquence** : destruction des CD4+ → immunodépression → SIDA (infections opportunistes, cancers).
- **HTLV** : oncogène (leucémies).

## MODULE III : MYCOLOGIE GÉNÉRALE

### 1. GÉNÉRALITÉS SUR LES CHAMPIGNONS

- **Définition** : eucaryotes, hétérotrophes (se nourrissent de matière organique), paroi à base de chitine/glucanes.
- **Formes principales** :
    - **Levures** : unicellulaires, reproduction par bourgeonnement (ex. *Candida*, *Cryptococcus*).
    - **Filamenteux (moisissures)** : formés de filaments (hyphes) cloisonnés ou non, formant un **mycélium**. Reproduction par spores.
    - **Dimorphes** : forme filamenteuse à 25°C, forme levure à 37°C (dans les tissus). (Ex. *Histoplasma*, *Sporothrix*).
- **Nutrition et métabolisme** : aérobies (plupart), température de croissance 20‑25°C (pathogènes à 37°C).
- **Reproduction asexuée** (intérêt diagnostique) :
    - **Spores endogènes** : *sporangiospores* (dans un sporange).
    - **Spores exogènes** : *conidies* (sur un conidiophore), *blastospores* (bourgeonnement), *arthrospores* (fragmentation), *chlamydospores* (paroi épaisse).

### 2. CLASSIFICATION DES MYCOSES (par site d’infection)

#### 2.1. Mycoses superficielles (peau, phanères)

- **Dermatophytoses** (filaments)
    - **Agents** : *Microsporum*, *Trichophyton*, *Epidermophyton* (kératinophiles).
    - **Sources** : anthropophile (homme), zoophile (animal), tellurique (sol).
    - **Clinique** : herpès circiné, intertrigos (grands plis, pied d’athlète), teignes, onyxis.
    - **Diagnostic** : examen direct des squames/cheveux/ongles (KOH) : filaments et arthrospores. Culture sur Sabouraud.

- **Malasseziose (Pityriasis versicolor)** (levure)
    - **Agent** : *Malassezia furfur* (lipophile).
    - **Clinique** : macules hypochromiques squameuses (cou, thorax).

- **Piedras** (nodosités sur poils)
    - **Noire** (*Piedraia hortae*) : nodules durs et foncés (tropiques).
    - **Blanche** (*Trichosporon beigelii*) : nodules mous et clairs (périnée, aisselles).

#### 2.2. Mycoses sous‑cutanées (inoculation traumatique)

- **Mycétomes**
    - **Définition** : tuméfaction + fistules + **grains**.
    - **Eumycétome** (fongique) : grains avec filaments larges (ex. *Madurella*).
    - **Actinomycétome** (bactérien) : grains avec filaments fins (<1 µm) (ex. *Nocardia*).

- **Chromoblastomycose**
    - **Agents** : champignons noirs dimorphes.
    - **Clinique** : lésions verruqueuses (jambes). Diagnostic : « cellules fumigatoires ».

- **Sporotrichose**
    - **Agent** : *Sporothrix schenckii* (dimorphe).
    - **Clinique** : forme lymphangitique (nodules en « chapelet »).
    - **Diagnostic** : culture +++.

- **Mucormycoses tropicales**
    - *Basidiobolus* : lésions dermo‑hypodermiques.
    - *Conidiobolus* : rhinofaciose (déformation du nez en « groin de tapir »).

#### 2.3. Mycoses profondes (souvent opportunistes)

- **Candidoses**
    - **Agent** : *Candida albicans* (levure commensale).
    - **Facteurs favorisants** : ATB, immunodépression, diabète.
    - **Clinique** : superficielle (muguet, onyxis avec **périonyxis**), profonde (septicémies).

- **Cryptococcose**
    - **Agent** : *Cryptococcus neoformans* (levure encapsulée).
    - **Diagnostic** : encre de Chine (visualise la capsule).
    - **Clinique** : pulmonaire, puis neuroméningée (méningo‑encéphalite) chez l’immunodéprimé.

- **Pneumocystose**
    - **Agent** : *Pneumocystis jirovecii*.
    - **Clinique** : pneumopathie chez l’immunodéprimé (SIDA, CD4 <200). Diagnostic sur LBA.

- **Histoplasmose**
    - **Agents** : *Histoplasma capsulatum* (américain) et *H. duboisii* (africain). Champignons dimorphes.
    - **Clinique** : formes pulmonaires, cutanées, osseuses.

- **Aspergilloses**
    - **Agent** : *Aspergillus* (filamenteux). Spores inhalées.
    - **Clinique** : allergique, aspergillome (boule fongique), invasive.
    - **Aflatoxines** : produites par *A. flavus* (hépatocarcinogène).

- **Désinfectants** : sur surfaces inertes, ne tuent pas forcément les spores.
- **Antiseptiques** : sur tissus vivants.

### 6. ANTIBIOTIQUES ET ANTIBIORÉSISTANCE

#### 6.1. Classification
- **Par cible** : paroi, membrane, synthèse protéique (ribosome 30S/50S), synthèse des acides nucléiques.
- **Par effet** : **bactéricides** (tuent) vs **bactériostatiques** (inhibent la croissance).

#### 6.2. Résistance aux antibiotiques
- **Types** : naturelle / acquise.
- **Mécanismes biochimiques** : inactivation enzymatique, modification de la cible, diminution de perméabilité, efflux actif.
- **Bases génétiques** :
    - **Mutation chromosomique** (sélection).
    - **Transfert de plasmides** (conjugaison) – dissémination rapide.
    - **Transposons** (« gènes sauteurs ») : intégration dans plasmide ou chromosome.

#### 6.3. Évaluation de l’efficacité
- **CMI** (concentration minimale inhibitrice) : empêche la croissance.
- **CMB** (concentration minimale bactéricide) : tue les bactéries.
- **Méthodes** : diffusion (antibiogramme de Kirby‑Bauer) – qualitative ; dilution (milieu liquide ou solide) – quantitative (CMI, CMB).

#### 6.4. Associations d’antibiotiques
- **Pourquoi ?** Synergie, prévention de la résistance (ex. tuberculose), élargissement du spectre, traitement probabiliste.
- **Résultats** : synergie, antagonisme, indifférence.

#### 6.5. Prévention de l’antibiorésistance (3 volets, 10 messages)
1. **Prévenir les infections** : vacciner, transmission croisée, limiter les invasifs.
2. **Mieux utiliser les ATB** : bonnes modalités, savoir dire non, bon traitement initial, savoir arrêter.
3. **Savoir refuser les ATB** : réévaluer à 48 h, traiter l’infection (pas la colonisation), infections bactériennes uniquement.

## MODULE II : VIROLOGIE GÉNÉRALE

### 1. GÉNÉRALITÉS SUR LES VIRUS

#### 1.1. Définition et structure
- **Définition** : agents infectieux **acellulaires**, possédant un seul type d’acide nucléique (ADN ou ARN), parasite intracellulaire obligatoire.
- **Structure (virion)** :
    - **Acide nucléique** : ADN ou ARN, simple brin (sb) ou double brin (db), linéaire/circulaire, segmenté.
        - **ARN positif** : directement traduit en protéines.
        - **ARN négatif** : doit être transcrit en ARN+.
    - **Capside** : coque protéique (protomères). Protège l’AN, antigénique. Symétries : **icosaédrique**, **hélicoïdale**, **complexe**.
    - **Nucléocapside** = AN + capside.
    - **Enveloppe** : bicouche lipidique (provenant de la cellule hôte) + glycoprotéines (gp) virales. Rôle : attachement, fusion, antigénicité. Sensible aux solvants.

#### 1.2. Classification
- Critères : type d’AN, symétrie, enveloppe, taille, hôte, mode de transmission, famille.

#### 1.3. Cycle de multiplication virale
1. **Adsorption** (attachement à un récepteur)
2. **Pénétration** (translocation, fusion, endocytose)
3. **Décapsidation** (libération de l’AN)
4. **Réplication/transcription** (synthèse des composants viraux)
5. **Assemblage** (auto‑assemblage)
6. **Libération** (par lyse ou bourgeonnement)

#### 1.4. Transmission et évolution
- **Modes** : aérienne, vectorielle (arbovirus), féco‑orale, contact direct (sang, sexe, salive), mère‑enfant.
- **Évolution** :
    - **Aiguë** : courte (grippe, Ebola).
    - **Persistante/chronique** : réplication continue (VIH, VHB, VHC).
    - **Latente** : virus présent mais non répliquant, réactivation possible (HSV, VZV).
    - **Lente** : longue incubation puis évolution fatale (prions).

### 2. ÉTUDE DES VIRUS PATHOGÈNES

#### 2.1. Arbovirus (transmis par arthropodes)
- **Exemples** : fièvre jaune (*Flavivirus*), dengue (*Flavivirus*), chikungunya (*Togaviridae*), fièvre de la Vallée du Rift (*Bunyaviridae*).
- **Tableaux** : fièvres hémorragiques, syndromes algo‑fébriles, encéphalites.

#### 2.2. Filovirus (Ebola, Marburg)
- **Caractéristiques** : ARNsb, enveloppé, filamentaire. Réservoir : chauve‑souris.
- **Pathologie** : fièvre hémorragique grave. Transmission interhumaine par liquides biologiques.
- **Souches Ebola** : Zaïre (ZEBOV, +virulente), Soudan, Bundibugyo, Côte d’Ivoire.

#### 2.3. Virus de la grippe (*Orthomyxoviridae*)
- **Caractéristiques** : ARN segmenté, enveloppé. Protéines de surface : **hémagglutinine (HA)** (attachement) et **neuraminidase (NA)** (libération).
- **Variabilité** : **glissement** (mutations ponctuelles, épidémies) et **cassure** (réassortiment génétique, pandémies).
- **Types** : A (pandémies), B, C.

#### 2.4. Virus de la rougeole et des oreillons (*Paramyxoviridae*)
- **Rougeole** : ARN, enveloppé. Transmission aérienne. Immunité durable.
- **Oreillons** : ARN, enveloppé. Transmission aérienne. Complications : orchite, méningite.

#### 2.5. Entérovirus (*Picornaviridae*)
- **Caractéristiques** : ARN, nus. Transmission féco‑orale.
- **Poliovirus** : maladie « iceberg » (peu de paralysies). Vaccin.
- **Coxsackies A/B, Echovirus** : syndromes variés (respiratoire, méningite, mains‑pieds‑bouche…).

#### 2.6. Herpesviridae
- **Caractéristiques** : ADNdb, enveloppé. Propriété clé : **latence**.
- **Alpha‑herpesvirinae** : HSV1 (oral), HSV2 (génital), VZV (varicelle, zona). Latence dans les ganglions nerveux.
- **Beta‑herpesvirinae** : CMV (mononucléose, infections congénitales), HHV6 (roséole). Latence dans monocytes/macrophages.
- **Gamma‑herpesvirinae** : EBV (mononucléose infectieuse, lymphomes), HHV8 (sarcome de Kaposi). Latence dans lymphocytes B.

#### 2.7. Virus de la rage (*Rhabdoviridae*)
- **Caractéristiques** : ARN, enveloppé, forme d’obus. Très fragile.
- **Transmission** : morsure/griffure d’animal infecté (zoonose). Virus neurotrope.
- **Prévention** : vaccination animale et humaine (après exposition).

#### 2.8. Virus des hépatites
| Virus | Famille | Génome | Transmission | Chronicité | Prévention |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **VHA** | Picornavirus | ARN nu | Féco‑orale | Non | Hygiène, vaccin |
| **VHB** | Hepadnavirus | ADN enveloppé | Sang, sexe, mère‑enfant | Oui (cirrhose, CHC) | Vaccin |
| **VHC** | Flavivirus | ARN enveloppé | Sang +++ | Oui (cirrhose, CHC) | Pas de vaccin |
| **VHD** | Virus défectif | ARN enveloppé | Sang, sexe | Oui (si co‑infection VHB) | Vaccin VHB |
| **VHE** | Calicivirus | ARN nu | Féco‑orale | Non | Hygiène |

#### 2.9. Papillomavirus (HPV) (*Papovaviridae*)
- **Caractéristiques** : ADNdb, nu. Tropisme cutanéo‑muqueux.
- **Types** :
    - **Non oncogènes** : verrues, condylomes acuminés (« crêtes de coq »).
    - **Oncogènes** (16, 18, 31, 33) : cancer du col de l’utérus, cancer anal.
- **Transmission** : contact cutané, IST+++.

#### 2.10. Rétrovirus (VIH, HTLV)
- **Caractéristiques** : ARN, enveloppé. Possèdent une **transcriptase inverse** (ARN → ADN).
- **VIH (virus de l’immunodéficience humaine)** :
    - **Structure** : gp120 (attachement au CD4), gp41 (fusion), transcriptase inverse.
    - **Cibles** : lymphocytes T CD4+ via récepteur CD4 et corécepteurs (CCR5, CXCR4).
    - **Cycle** : fixation → fusion → décapsidation → transcription inverse → intégration → réplication → assemblage → bourgeonnement.
    - **Conséquence** : destruction des CD4+ → immunodépression → SIDA (infections opportunistes, cancers).
- **HTLV** : oncogène (leucémies).

## MODULE III : MYCOLOGIE GÉNÉRALE

### 1. GÉNÉRALITÉS SUR LES CHAMPIGNONS

- **Définition** : eucaryotes, hétérotrophes (se nourrissent de matière organique), paroi à base de chitine/glucanes.
- **Formes principales** :
    - **Levures** : unicellulaires, reproduction par bourgeonnement (ex. *Candida*, *Cryptococcus*).
    - **Filamenteux (moisissures)** : formés de filaments (hyphes) cloisonnés ou non, formant un **mycélium**. Reproduction par spores.
    - **Dimorphes** : forme filamenteuse à 25°C, forme levure à 37°C (dans les tissus). (Ex. *Histoplasma*, *Sporothrix*).
- **Nutrition et métabolisme** : aérobies (plupart), température de croissance 20‑25°C (pathogènes à 37°C).
- **Reproduction asexuée** (intérêt diagnostique) :
    - **Spores endogènes** : *sporangiospores* (dans un sporange).
    - **Spores exogènes** : *conidies* (sur un conidiophore), *blastospores* (bourgeonnement), *arthrospores* (fragmentation), *chlamydospores* (paroi épaisse).

### 2. CLASSIFICATION DES MYCOSES (par site d’infection)

#### 2.1. Mycoses superficielles (peau, phanères)

- **Dermatophytoses** (filaments)
    - **Agents** : *Microsporum*, *Trichophyton*, *Epidermophyton* (kératinophiles).
    - **Sources** : anthropophile (homme), zoophile (animal), tellurique (sol).
    - **Clinique** : herpès circiné, intertrigos (grands plis, pied d’athlète), teignes, onyxis.
    - **Diagnostic** : examen direct des squames/cheveux/ongles (KOH) : filaments et arthrospores. Culture sur Sabouraud.

- **Malasseziose (Pityriasis versicolor)** (levure)
    - **Agent** : *Malassezia furfur* (lipophile).
    - **Clinique** : macules hypochromiques squameuses (cou, thorax).

- **Piedras** (nodosités sur poils)
    - **Noire** (*Piedraia hortae*) : nodules durs et foncés (tropiques).
    - **Blanche** (*Trichosporon beigelii*) : nodules mous et clairs (périnée, aisselles).

#### 2.2. Mycoses sous‑cutanées (inoculation traumatique)

- **Mycétomes**
    - **Définition** : tuméfaction + fistules + **grains**.
    - **Eumycétome** (fongique) : grains avec filaments larges (ex. *Madurella*).
    - **Actinomycétome** (bactérien) : grains avec filaments fins (<1 µm) (ex. *Nocardia*).

- **Chromoblastomycose**
    - **Agents** : champignons noirs dimorphes.
    - **Clinique** : lésions verruqueuses (jambes). Diagnostic : « cellules fumigatoires ».

- **Sporotrichose**
    - **Agent** : *Sporothrix schenckii* (dimorphe).
    - **Clinique** : forme lymphangitique (nodules en « chapelet »).
    - **Diagnostic** : culture +++.

- **Mucormycoses tropicales**
    - *Basidiobolus* : lésions dermo‑hypodermiques.
    - *Conidiobolus* : rhinofaciose (déformation du nez en « groin de tapir »).

#### 2.3. Mycoses profondes (souvent opportunistes)

- **Candidoses**
    - **Agent** : *Candida albicans* (levure commensale).
    - **Facteurs favorisants** : ATB, immunodépression, diabète.
    - **Clinique** : superficielle (muguet, onyxis avec **périonyxis**), profonde (septicémies).

- **Cryptococcose**
    - **Agent** : *Cryptococcus neoformans* (levure encapsulée).
    - **Diagnostic** : encre de Chine (visualise la capsule).
    - **Clinique** : pulmonaire, puis neuroméningée (méningo‑encéphalite) chez l’immunodéprimé.

- **Pneumocystose**
    - **Agent** : *Pneumocystis jirovecii*.
    - **Clinique** : pneumopathie chez l’immunodéprimé (SIDA, CD4 <200). Diagnostic sur LBA.

- **Histoplasmose**
    - **Agents** : *Histoplasma capsulatum* (américain) et *H. duboisii* (africain). Champignons dimorphes.
    - **Clinique** : formes pulmonaires, cutanées, osseuses.

- **Aspergilloses**
    - **Agent** : *Aspergillus* (filamenteux). Spores inhalées.
    - **Clinique** : allergique, aspergillome (boule fongique), invasive.
    - **Aflatoxines** : produites par *A. flavus* (hépatocarcinogène).
